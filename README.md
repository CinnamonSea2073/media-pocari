# media-pocari

React + Vite で `pocari.svg` を画面いっぱいに表示するだけのシンプルなサイトです。

## 必要環境

- Node.js (推奨: 18 以降)
- npm

## セットアップ

```bash
npm install
```

## 開発サーバー起動

```bash
npm run dev
```

ブラウザで表示される URL (通常 `http://localhost:5173`) を開くと、背景色 `#004D9D` に `pocari.svg` が画面いっぱいに表示されます。

`pocari.svg` はプロジェクト直下の `public/pocari.svg` に配置してください。
